# TICKET-057 — Shuffle answer options on AI-generated questions

**Status:** Done  
**Type:** Bug  
**Priority:** Medium

## Problem

When questions are generated by the AI, the correct answer is always placed first in the list of answer options. This makes it trivially easy for players to spot the pattern and guess correctly without reading the question. The answer options should be shuffled so the correct answer appears in a random position across the generated set.

## Acceptance criteria

- [x] Answer options for AI-generated questions are shuffled before being saved to the database
- [x] The correct answer is not consistently in the first position across multiple generated questions
- [x] Shuffling applies to both `MULTIPLE_CHOICE` and `IMAGE` type generated questions
- [x] Existing manually created questions are unaffected

## Resolution

In `server/src/routes/quiz.ts`, after parsing Claude's response, each question's `answerOptions` array is now shuffled with a random sort before being passed to `buildRelations` and saved. Existing manually created questions go through a separate code path and are unaffected.
