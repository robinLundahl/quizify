# Backlog

This folder tracks all upcoming tasks and tickets for the quiz project.

## Workflow

1. **Create** a new ticket as `TICKET-NNN.md` (increment the number from the last one)
2. **Work** the ticket — update `Status` to `In Progress` when you start
3. **Archive** — when done, move the file to `backlog/archive/`

## Ticket template

```markdown
# TICKET-NNN: Short Title

**Status:** Open
**Created:** YYYY-MM-DD
**Priority:** High | Medium | Low

## Description
What needs to be done and why.

## Acceptance Criteria
- [ ] Specific, testable outcome

## Notes
Any extra context, links, or decisions.
```

## Folders

- `backlog/` — open and in-progress tickets
- `backlog/archive/` — completed tickets (history)
