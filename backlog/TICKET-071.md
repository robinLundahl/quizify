# TICKET-071 — Implement sales analytics for Pro creators

**Status:** Open  
**Type:** Feature  
**Priority:** Low

## Goal

Give Pro creators an analytics view inside their Earnings tab showing per-quiz performance highlights. This is a Pro-only feature — free creators see the standard earnings view without analytics. Balance and payout history are available to all users via TICKET-068.

## Acceptance criteria

- [ ] Analytics section is visible inside the Earnings tab for Pro users only
- [ ] Displays top 3 quizzes by average rating
- [ ] Displays top 3 quizzes by number of sales
- [ ] Free users see the analytics section locked with an upgrade prompt
