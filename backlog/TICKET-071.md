# TICKET-071 — Implement sales analytics for Pro creators

**Status:** Open  
**Type:** Feature  
**Priority:** Low

## Goal

Give Pro creators a sales analytics view in their dashboard showing earnings, sales volume, and per-quiz performance. This is a Pro-only feature — free creators do not have access.

## Acceptance criteria

- [ ] Analytics section is visible in the dashboard for Pro users only
- [ ] Displays total earnings (all time), pending balance, and available balance
- [ ] Displays per-quiz breakdown: number of sales, revenue, average rating
- [ ] Displays a sales over time chart (daily or weekly)
- [ ] Free users see the analytics section locked with an upgrade prompt
